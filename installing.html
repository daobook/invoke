基本安装¶

获得 Invoke 的推荐方法是通过 pip 安装最新的稳定版

$ pip install invoke

我们目前支持 Python 2.7 和 Python 3.4+。仍然使用 Python 2.6 或更早的版本，或 3.3 或更早的版本的用户，被敦促升级。

只要你有支持的 Python 解释器，就没有其他依赖性。Invoke 是纯 Python 的，在它的源码树中包含了少数依赖的副本。

请参阅 这篇博文，了解我们决定将依赖关系供应商化的背景。
